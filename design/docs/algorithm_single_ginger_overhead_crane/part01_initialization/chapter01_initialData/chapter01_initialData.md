
# Ввод исходных данных для опорного однобалочного крана мостового типа
  
Вне зависимости от исходных данных однобалочный мостовой кран (single ginger overhead crane) всегда имеет обязательно:

* одну или несколько грузовых талей (тельферов) с механизмом подъёма;
* мост крана, установленный на концевые балки или балансиры.

**Параметры механизма подъёма (hoist):**

* грузоподъёмность механизма (load), т - *число с плавающей запятой*;
* высота подъёма механизма (lifting height), м - *число с плавающей запятой*;
* тип грузозахватного органа механизма подъёма (lifting device) - *выбор значения из списка*:  
  * крюковая подвеска (hook block);
  * съёмный электрогидравлический грейфер (electro-hydraulic grab), подвешиваемый на крюковой подвеске;
  * съёмный электромагнит (electromagnet), подвешиваемый на крюковой подвеске;
* номинальная скорость подъёма механизма (rated travelling hoist speed - $\nu_{hmax}$), м/мин - *число с плавающей запятой*;
* замедленная скорость подъёма механизма (slow travelling hoist speed - $\nu_{hCS}$), м/мин - *число с плавающей запятой*;
* режим работы механизма подъёма (hoist group) согласно ГОСТ 34017-2016 - *выбор значения из списка*:
  * M1 - M8;
* продолжительность включения (ПВ) механизма подъёма (lifting duration) согласно ГОСТ IEC 60034-1-2014 - *выбор значения из списка*:
  * 15%,
  * 25%,
  * 40%,
  * 60%,
  * 100%;
* система управления приводом (hoist control system) - *выбор значения из списка*:
  * частотное регулирование - Variable Frequency Drive (VFD),
  * релейно-контакторная система - Relay-Contactor Control System,
  * тиристорно-дроссельная - Thyristor-Choke Control System;
* тип привода механизма подъёма (lifting mtchanism drive type) - *выбор значения из списка*:
  * HD1 - нет замедленной скорости,
  * HD2 - постоянная замедленная скорость,
  * HD3 - постоянная замедленная скорости до отрыва груза от земли,
  * HD4 - система бесступенчатого управления переменной скоростью,
  * HD5 - после предварительного натяжения разгон до выбранной скорости;
* тип поднимаемого груза (type of lifted load) - *выбор значения из списка*:
  * безопасный (safe),
  * опасный (отравляющие, взрывчатые, горячие грузы, расплавленный металл и т.п.) (dangerous);

**Параметры механизма передвижения грузовой тали (the trolley running mechanism):**

* номинальная скорость передвижения тали (rated travelling trolley speed - $\nu_{hmax}$), м/мин - *число с плавающей запятой*;
* замедленная скорость передвижения тали (slow travelling trolley speed - $\nu_{hCS}$), м/мин - *число с плавающей запятой*;
* режим работы механизма передвижения тали (trolley group) согласно ГОСТ 34017-2016 - *выбор значения из списка*:
  * M1 - M8;
* система управления приводом (trolley control system) - *выбор значения из списка*:
  * частотное регулирование - Variable Frequency Drive (VFD),
  * релейно-контакторная система - Relay-Contactor Control System,
  * тиристорно-дроссельная - Thyristor-Choke Control System;
* тип токоподвода к грузовой тали (trolley power system) - *выбор значения из списка*:
  * фестонный (festoon system),
  * траковый (energy chain),
  * троллейный (conductor bar).

**Параметры механизма передвижения крана (the bridge running mechanism):**

* номинальная скорость передвижения моста крана (rated travelling bridge speed - $\nu_{hmax}$), м/мин - *число с плавающей запятой*;
* замедленная скорость передвижения моста крана (slow travelling bridge speed - $\nu_{hCS}$), м/мин - *число с плавающей запятой*;
* режим работы механизма передвижения моста крана (crane drive group) согласно ГОСТ 34017-2016 - *выбор значения из списка*:
  * M1 - M8;
* продолжительность включения (ПВ) механизма (bridge movement duration) согласно ГОСТ IEC 60034-1-2014 - *выбор значения из списка*:
  * 15%,
  * 25%,
  * 40%,
  * 60%,
  * 100%;
* система управления приводом (bridge control system) - *выбор значения из списка*:
  * частотное регулирование - Variable Frequency Drive (VFD),
  * релейно-контакторная система - Relay-Contactor Control System,
  * тиристорно-дроссельная - Thyristor-Choke Control System;
* тип токоподвода к крану (crane power system) - *выбор значения из списка*:
  * фестонный (festoon system),
  * кабельный барабан (cable reel),
  * траковый (energy chain),
  * троллейный (conductor bar);
* тип механизма передвижения (bridge drive type system) - *выбор значения из списка*:
  * развёрнутая схема (detailed bridge drive),
  * мотор-редуктор (gearmotor);
* схема привода передвижения крана (bridge drive diagram) - *выбор значения из списка*:
  * центральный (общий) привод (central drive),
  * раздельный привод (separate drive);
* система синхронизации механизма передвижения крана (bridge control system of synchronous movement) - *выбор значения из списка*:
  * присутствует (present),
  * отсутствует (absent);
* тип подкранового пути (type crane rail) - *пустая или заполненная строка*;
* длина подкранового пути (crane rail length), м  - *число с плавающей запятой или пустое значение*.

**Общие параметры крана (general crane parameters):**

* исполнение крана - *выбор значения из списка*:
  * общепромышленный (industrial purpose),
  * металлургический (metallurgical purpose),
  * специальный (special purpose),
  * судовой (marine purpose);
* взрыво-пожаробезопасное исполнение крана (explosion-fire-safe crane purpose) - *выбор значения из списка*:
  * общепромышленное (industrial purpose),
  * пожарозащищённое (fire-safe purpose),
  * взрывозащищённое (explosion-safe purpose);
* маркировка пожаро-/взрывоопасной среды эксплуатации крана (marking of fire/explosion hazardous operating environment) - *заполненная или пустая строка*;
* режим работы крана согласно ГОСТ 34017-2016 (duty class) - *выбор значения из списка*:
  * A0-A11;
* климатическое исполнение и категория размещения крана согласно ГОСТ 15150-69 (climatic design and placement category crane) - *выбор значения из списка*:
  * У1,
  * У2,
  * У3,
  * У5,
  * ХЛ1,
  * ХЛ2,
  * ХЛ3,
  * УХЛ4,
  * УХЛ4.1,
  * УХЛ4.2,
  * О4,
  * О4.1,
  * О4.2,
  * Т5,
  * ТС2,
  * В3,
  * В3.1,
  * В4.1,
  * ОМ.4,
  * В5;
* ветровой район расположения крана (crane wind area) - *выбор значения из списка*:
  * 0,
  * I,
  * II,
  * III,
  * IV,
  * V,
  * VI,
  * VII,
  * морской;
* максимальная температура эксплуатации (max use temperature), &deg;C - *целое число*;
* минимальная температура эксплуатации (min use temperature), &deg;C - *целое число*;
* основной вид управления краном (basic crane control) - *выбор значения из списка*:
  * из кабины (crane cab);
  * с подвесного пульта (pendant control);
  * радиоуправление (remote control);
  * из кабины + радиоуправление (cab/remote control);
* место расположения кабины (cab location) - *выбор значения из списка*:
  * край моста (bridge edge),
  * центр пролёта моста (bridge span center),
  * грузовая тележка (crane trolley);
  * пустое значение (None value);
* количество одинаковых грузовых тележек на кране, шт (identical hoists volume) - *целое число*;
* максимальная масса крана (max crane mass), т - *число с плавающей запятой или пустое значение*.
* Допускаемое давление колеса на подкрановый путь (max wheel load), кН - *число с плавающей запятой или пустое значение*.

**Габаритные размеры крана:**

* пролёт крана (span), м - *число с плавающей запятой*;
* приближение грузозахватного органа к левому краю (l1 на рисунке) (left edge approach lifting device), м - *число с плавающей запятой или пустое значение*;
* приближение грузозахватного органа к правому краю (l2 на рисунке) (right edge approach lifting device), м - *число с плавающей запятой или пустое значение*;
* вертикальное расстояние от подкранового пути до грузозахватного органа (vertical distance from crane rail to lifting device), м - *число с плавающей запятой или пустое значение*;
* максимальная база крана, м;
* максимальное вертикальное расстояние от подкранового пути до верхней точки крана (maximum vertical distance from the crane rail to the top of the crane), м - *число с плавающей запятой или пустое значение*;
* максимальная база крана (max crane base), м - *число с плавающей запятой или пустое значение*;
* максимальная ширина крана (max width crane), м - *число с плавающей запятой или пустое значение*.
